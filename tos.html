What you can copy

The joys of open-source projects, here is what you can copy when you make a fork.

  • The content that is hosted on this site
  • Any 7Zeb games and applications* (*Except for some, please learn more at their Github Repository*)
  • Any HTML or JavaScript code hosted on this site* (*Except for some, the ones that you can't will have a message on either a comment (if javascript) or the footer in the HTML file.)

About the CSS

You CAN NOT copy any CSS files from this site, if you make a fork of this repository, you MUST remove any CSS files, however there are exceptions to this, and they are listed below.

  1. If the CSS is from LegacyUI (which was from August 2024 - April 2, 2025)
  2. If the version of NewUI is an old version (for example: if NewUI 3 is being used on the current site, you can you can use NewUI 2 for your project.
  3. If I (7Zeb) have given you permission to use NewUI in your project.
  4. If the CSS file contains a comment stating you are allowed to copy it (example: pcbuild25.css)
  5. If you are improving NewUI (a pull request must be made within a week for this exception to apply)
  6. If you are not publishing this site online, and are only hosting it locally or having a local copy of it (you still CAN NOT have it on a Github Repository)

Please follow these rules on the CSS and you should be fine.
